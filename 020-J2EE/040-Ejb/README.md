# Enterpirse Java Bean (EJB)
Enterprise Bean is a server-side component that encapsulates business logics. In a J2EE Container, the EJB can be reused by different application.

## Type
* Session Bean
* Message Driven Bean 

## Access Type
* Local 
* Remote
